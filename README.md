# Unity Gameplay Scripts

Conjunto de scripts desenvolvidos em **Unity (C#)** com foco em lógica de gameplay.
Este projeto faz parte do meu **portfólio pessoal**, demonstrando organização de código,
boas práticas e implementação de sistemas comuns em jogos 2D.

## Funcionalidades implementadas
- Movimento do jogador com caminhada, corrida e rolagem
- Sistema de animação integrado ao movimento
- Interação com NPCs baseada em proximidade
- Sistema de diálogo com:
  - Suporte a múltiplos idiomas
  - Efeito de digitação (typewriter)
  - Estrutura modular para expansão

## Estrutura do projeto
- **Player**  
  Controle de movimento, corrida, rolagem e estados do jogador.
- **PlayerAnim**  
  Controle de animações baseado nos estados do Player.
- **NPC**  
  Detecção de proximidade e ativação de diálogos.
- **Dialogue**  
  Sistema de diálogo reutilizável e escalável.

## Tecnologias
- Unity (2D)
- C#
- Animator Controller
- Física 2D (Rigidbody2D)

## Observações
Projeto em desenvolvimento contínuo, criado com foco em aprendizado prático e
aprimoramento de arquitetura de scripts em Unity.
